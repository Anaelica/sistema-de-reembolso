# 💸 Sistema de Controle de Despesas

Projeto desenvolvido com JavaScript para gerenciamento de despesas, permitindo adicionar, remover e calcular gastos dinamicamente.

---

## 📌 Funcionalidades

-  Adicionar despesas
-  Selecionar categorias
-  Formatação automática de moeda BRL
-  Contador de despesas
-  Soma automática dos gastos
-  Remover despesas
-  Confirmação antes de remover itens
-  Atualização dinâmica da interface

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🧠 Conceitos Trabalhados

- Manipulação do DOM
- Eventos JavaScript
- Funções
- Objetos
- Loops
- Template Strings
- Formatação de moeda
- Delegação de eventos
- Tratamento de erros com try/catch
